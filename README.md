## Available Scripts

In the project directory, you can run:

### `npm start`

Runs in development mode on local server.

### `npm run build`

Builds the project

### `npm run start`

Serves the application in production mode.\

### `npm run heroku-postbuild`

Equal to `npm run build` but for Heroku-Service\
